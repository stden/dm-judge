1013. Гадание на Словах

Всем давно известно, что гадальные автоматы зачастую придумывают истории для того, кто бросил в нее монетку. Конечно, бывает иногда и так, что гадание сбывается.

Компания «Mocrosoft–Fortune» решила разработать новый гадальный автомат. Беднягу просят сказать два слова A и B, затем автомат из букв этого слова составляет третье слово - C. Причем буквы из слова A и из слова B встречаются в слове С в том же порядке, что и в самих словах A, B. Гадальная машина берет буквы из A и B в таком порядке, что буквы в новом слове–гадании C получаются максимально упорядоченными. То есть две рядом стоящие буквы в C идут в убывающем порядке как можно реже. Помогите программистам компании «Mocrosoft–Fortune».

Формализуем задачу

Вам даны два слова A и B, состоящие из прописных букв латинского алфавита {a,b,…,z}. В слово A необходимо вставить буквы слова B так, чтобы при этом порядок следования букв слова B сохранился. При этом, количество убывающих пар двух рядом стоящих букв полученного слова C было бы минимально.

Примечание: Для примера возмем A=«opq» и B=«leti», тогда оптимальное гадание получается C= «LopqETI». Здесь «lopq» и «et» - идут по неубыванию в соответствии с алфавитом, но «qe» и «ti» - две пары в убывающем порядке.

Формат входных данных:

Во входном файле на первой строке дано слово A, затем на второй строке дано слово B. Длина слов A и B находится в пределах от 1 до 100.

Формат выходных данных:

На первой строке выведите количество пар в новом слове C, которые идут в убывающем порядке. Это количество должно быть минимально возможным.

На второй строке выведите полученное слово C. Для наглядности, буквы слова B выведите заглавными буквами. Если правильных решений несколько, то выведите любое.

function check_one_test( $A, $B, $C, $sizeJury, $sizeStudent ) {
  if($sizeJury != $sizeStudent) return false;
  if(strlen($C) != (strlen($A)+strlen($B))) return false;
  $B = strtoupper($B);
  $A = strtolower($A);
  for($k=$i=$j=0; $C[$k]; $k++)
    if($C[$k]>='a' && $C[$k]<='z') {
      if($C[$k]!=$A[$i] || $i>=strlen($A)){ return false; } else $i++;
    } else {
      if($C[$k]!=$B[$j] || $j>=strlen($B)){ return false; } else $j++;
    };
  $C = strtolower($C);
  for($i=$k=0; $C[$k+1]; $k++)
    if($C[$k]>$C[$k+1]) $i++;
  if($i!=$sizeJury) return false;
  return true;
}
