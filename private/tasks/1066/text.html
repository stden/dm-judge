1066. Постфиксная запись

Математическое выражение в инфиксной скобочной записи использует следующие знаки операций:

Приоритет

Операции

1

(  ) – скобки

2

& - взятие квадратного корня (унарная)

3

^ - возведение в степень

4

* - умножить

/ - деление

5

+ - плюс

– - минус

Входные данные:

В первой строке входного файла содержится количество тестов T. Далее идет T строк - по одной строке на тест. Каждый тест состоит из выражения в инфиксной форме.

Выходные данные:

Для каждого теста вывести постфиксную запись выражения и минимальную глубину стека для преобразования.

Если входное выражение неправильно записано, то в выходной файл выводится “Error!”. Неправильной является запись, в которой неправильно расставлены скобки, знаки операций или переменные.
