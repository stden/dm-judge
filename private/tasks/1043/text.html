1043. Логическая функция

Вам предстоит решить очень простую задачу: дана логическая функция, которая включает конъюнкцию (& - and), дизъюнкцию (| - or) и отрицание (! - not). Для еще большего упрощения вашей задачи скобки отсутствуют.

Необходимо вычислить значение функции для нескольких заданных наборов значений переменных.

Входные данные:

В первой строке входного файла содержится количество тестов T.

Каждый тест начинается со строки, в которой указана функция F (без пробелов), затем 2 пробела, затем количество наборов V. Все логические переменные обозначены прописными английскими буквами (A..Z).

В последующих V строках записаны наборы 0 и 1 – значения переменных, в следующем порядке: значение A, значение B, … значение Z.

Выходные данные:

Для каждого теста надо вывести V нулей или единиц (разделяя переводами строки) – значение F для заданного набора значений переменных.
