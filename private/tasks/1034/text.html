1034. Делимость

Дано N целых чисел (1≤N≤100). Определить можно ли из них выбрать несколько чисел таким образом, чтобы сумма выбранных чисел делилась на К (|K|≤100). Все числа по модулю не превосходят 215.

Входные данные:

В первой строке входного файла содержится количество тестов T. Далее идут T тестов.

Во в каждом тесте в первой строке через пробел находятся два числа: N и K. В последующих строках содержатся N чисел, среди которых нужно выбирать.

Выходные данные:

Для каждого теста программа должна выдать YES, если выбрать такую группу чисел возможно, и NO в противном случае.

Пример теста

Пример ответа

1

3 5

2 1 9

YES

Источник: Денис Назаров. Олимпиады школьников в Уфе.
