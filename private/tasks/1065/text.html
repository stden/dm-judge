1065. Префиксная запись

Определение:

префиксной формой записи называется выражение вида:

<префикс> ::= <знак_бинарной_операции><префикс><префикс> | <операнд> |

  <знак_унарной_операции><префикс>

<знак_бинарной_операции> ::= + | - | * | / | ^

<знак_унарной_операции> ::= C | S | L | K

<операнд> ::= a | b | c | … | z

Примеры: +a-bc, ^xy

  

Задание:

Проверить правильность записи выражений, представленных в префиксной форме.

 

Замечание:  Cx=cos(x) Lx=lg(x)  Kx=Öx

              Sx=sin(x)  ^xy=xy

Входные данные:

В первой строке входного файла содержится количество тестов T. Далее идет T строк - по одной строке на тест. Каждый тест состоит из выражения в префиксной форме.

Выходные данные:

Выходной файл должен состоять из T строк. В каждую строку файла нужно поместить сообщение TRUE, если выражение составлено верно, или FALSE  -  если неверно.
