1018. Рыцари круглого стола

На клетчатой доске расположено N замков, в каждом из которых живет  один рыцарь.

Король Артур хочет регулярно собирать своих вассалов за круглым столом для обсуждения важных вопросов. Вся беда в том, что рыцари очень заняты своими делами, поэтому хотят тратить на путь как можно меньше времени (рыцарь может переместиться за один ход на одну клетку вверх, вниз, вправо или влево, но не по диагонали).

Понятно, что угодить каждому рыцарю невозможно, всегда кто-то будет обижен. Ваша задача состоит в том, чтобы выбрать такое место для круглого стола, чтобы суммарный путь всех рыцарей был как можно меньше.

Формат входных данных:

Первая строка входного файла содержит одно натуральное число N – количество замков ().

Далее следует  строк, каждая из которых содержит два натуральных числа, разделённых пробелом – координаты -го замка ()

Формат выходных данных:

В выходном файле должна быть ровно одна строка, содержащая одно натуральное число – минимальный суммарный путь рыцарей.
