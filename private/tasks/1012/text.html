1012. Сундук

Темные рыцари черного владыки Саурона после смерти господина страшным заклинанием четырех стихий создали волшебный сундук для хранения кольца всевластия, дабы сохранить его до возвращения господина.

Ни один из ныне живущих магов не может вскрыть сундук силой своего заклинания. Кодовый замок сундука состоит из N колец. На каждом кольце есть одна насечка, а вокруг каждого кольца есть K делений (как на циферблате часов).

Вы – начинающий маг и способны за одно заклинание повернуть только одно кольцо на одно деление и лишь по часовой стрелке. Полузабытая легенда гласит, если все насечки будут указывать на одно и то же деление, то сундук откроется, а кольцо всевластия станет Вашим!

С каждым заклинанием Ваша магическая сила иссякает, что опасно – силы зла не дремлют… Сколько заклинаний Вы потратите, чтобы получить кольцо?

Формат входных данных:

В первой строчке входного файла записано два натуральных числа - количество колец  и количество делений  вокруг каждого из изображений кольца.

Во второй строке записано  целых чисел  в диапазоне - для каждого изображения кольца указан номер риски, на которую оно указывает (риски пронумерованы по часовой стрелке).

Формат выходных данных:

Минимальное количество заклинаний, которое потребуется Вам, чтобы повернуть все кольца на выбранное Вами деление  (). Выбор  остается за Вами…
