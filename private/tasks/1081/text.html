1081. Р-ичное умножение

Даны 2 числа в системе счисления с основанием Р. Найти их произведение.

 

Входные данные:

N

P1

(A11;A12…;A1m)

(B11;B12…;B1m)

……

PN

(AN1;AN2…;ANk)

(BN1;BN2…;BNk)

 

Комментарии:

0<N<100 – количество тестов

Aij, Bij  (где 1≤ j ≤m)  – значения j-тых разрядов (начиная со старшего) множителей в системе счисления с основанием Pi.

На i-м тесте – десятичные числа в диапазоне [0; Pi-1]. 0<m,k<100 

1 < Pi ≤ 180

 

Выходные данные:

(C11;C12…;C1q)

…

(CN1;CN2…;CNq)

 

Комментарии:

Выходной файл содержит N произведений соответствующих чисел из входных данных – числа, записанные в том же виде, что и множители.
