1000. A+B Problem

Требуется сложить два числа a и b. 0≤a,b≤109.

Входные данные:

Входной файл содержит несколько тестов. Первая строка входного файла содержит одно натуральное число N - количество тестов.

Далее идет N строк – сами тесты.

Каждый тест состоит из одной строки, содержащей два целых числа a и b, разделенных пробелом.

Выходные данные:

В каждой строке выходного файла вывести a+b для очередного теста.

Пример:

Пример входных данных

Пример выходных данных

3

2 2

3 5

14232 234254

4

8

248486

Внимание:

Данная задача предназначена для ознакомления с тестирующей системой и потому здесь приводится ее решение:

 

{ Решение задачи 1000 на языке Паскаль }
procedure solve;
var a,b : longint;
begin
  read(a,b);
  writeln(a+b);
end;

{ Общая для всех задач часть }
var i,n : longint;
begin
  { Входные данные следует скопировать в файл in.txt }
  assign(input,'in.txt'); reset(input);
  { Выходные данные будут выведены в файл out.txt }
  assign(output,'out.txt'); rewrite(output);
  {}
  read(n); { Считываем количество тестов }
  for i:=1 to n do solve;
end.
