1010. Экспонента

Положим N=ab. Найдите последнюю цифру числа N.

Входные данные

Первая строка входного файла содержит целое число a, вторая строка содержит целое число b. (1<a,b<10100).

Выходные данные

Выходной файл должен содержать последнюю цифру числа N..

Пример ответа
