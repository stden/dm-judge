Требуется найти подмассив с максимальной суммой элементов.

Входные данные:

В первой строке входного файла задаётся единственное число N (1≤N≤100). Следующие N строк содержат по N целых чисел, разделенных одним пробелом. (-104≤Aij≤104)

Выходные данные:

Выходной файл содержит искомый подмассив из нескольких строк и столбцов с максимальной суммой элементов. Числа в строке разделяются одним пробелом.
